# BCM Frontend

Frontend React app for Better Calories Measurement.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

The app will run on http://localhost:3000

## Features

- Photo capture using device camera
- Image upload functionality
- Real-time calorie analysis
- Nutrition breakdown display
- Responsive design

## Backend Integration

The frontend connects to the backend API running on port 5000. Make sure the backend is running before starting the frontend.
